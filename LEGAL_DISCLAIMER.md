## 1. 数据来源声明

### 1.1 信息采集方式

本小程序展示的夏令营信息来源于以下渠道：

1. **公开信息采集**：通过自动化技术从各高校官方网站公开的招生信息页面采集
2. **官方发布渠道**：采集各院校研究生院、招生办公室官方发布的公告
3. **用户反馈补充**：用户可通过"信息纠错"功能提交补充信息

### 1.2 信息使用原则

- 仅采集和展示**已公开发布**的招生信息
- 严格遵守各院校的robots.txt协议
- 控制采集频率，避免对院校服务器造成负担（每域名≤1请求/分钟）
- 仅展示信息摘要，完整信息引导用户访问院校官网

## 3. 合规措施

### 3.1 爬虫合规措施

为确保数据采集活动合法合规，我们采取以下措施：

| 措施 | 说明 |
|------|------|
| **遵守robots.txt** | 严格遵守各院校网站的robots.txt协议 |
| **限速采集** | 每域名采集频率≤1请求/分钟，避免对服务器造成压力 |
| **仅采集公开信息** | 不采集需要登录或付费才能访问的信息 |
| **信息摘要展示** | 仅展示标题、摘要等基本信息，引导用户访问原文 |
| **及时响应投诉** | 建立院校投诉处理机制，24小时内响应删除请求 |

### 3.2 数据保护措施

- 仅收集用户微信openid（匿名标识），不收集手机号、姓名等敏感信息
- 用户选择偏好数据加密存储
- 数据传输全程HTTPS加密
- 定期删除过期数据

### 3.3 用户权益保障

- 用户可随时删除账号及关联数据
- 用户可举报不准确信息
- 用户可取消订阅消息推送
- 用户数据仅用于提供服务，不向第三方出售

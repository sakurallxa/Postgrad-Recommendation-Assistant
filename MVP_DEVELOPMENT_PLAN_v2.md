# 保研信息助手小程序 - MVP核心功能开发计划（评审后修订版）

**文档版本**: v2.0  
**修订日期**: 2026-02-25  
**计划周期**: 6周（2026-02-25 至 2026-04-08）  
**修订说明**: 基于技术评审报告调整，延长开发周期，强化风险应对

---

## 1. 修订说明

### 1.1 主要调整内容

基于技术评审报告（v1.0）的结论，本次修订对原开发计划进行了以下调整：

| 调整项 | 原方案 | 修订后方案 | 原因 |
|--------|--------|-----------|------|
| **开发周期** | 4周 | 6周 | 技术栈复杂度高，4周不可行 |
| **功能范围** | 包含提醒功能 | 提醒功能延后至V2 | 优先保证核心查询功能 |
| **风险应对** | 简单提及 | 详细应对方案 | 3项致命风险必须处理 |
| **架构设计** | 基础版本 | 增加降级/熔断机制 | 提升系统稳定性 |
| **成本控制** | 未明确 | API调用上限+监控 | 防止成本失控 |

### 1.2 评审结论整合

**已采纳的评审建议**：
- ✅ 延长开发周期至6周
- ✅ 提醒功能延后至V2版本
- ✅ 添加数据库复合索引
- ✅ 实现接口限流中间件
- ✅ DeepSeek API设置调用上限
- ✅ 爬虫添加降级方案
- ✅ 实现日志脱敏

**待V2阶段实施的改进**：
- ⏳ 引入消息队列解耦
- ⏳ 容器化部署
- ⏳ 完善监控告警体系
- ⏳ 微服务拆分

## 2. 开发目标（修订后）

完成MVP版本核心功能（20个功能点），确保产品能够：

1. **用户认证**：通过微信登录并保存院校/专业选择
2. **信息查询**：展示精准的夏令营信息列表（支持多维度筛选）
3. **数据更新**：通过爬虫服务自动采集夏令营信息（含AI提取）
4. **系统稳定**：具备降级能力和成本控制机制

**明确不包含在MVP的功能**：
- ❌ 微信订阅消息提醒（延后至V2）
- ❌ 定时推送服务（延后至V2）
- ❌ 用户提醒管理（延后至V2）

model University {
  id          String   @id @default(uuid())
  name        String   @unique
  code        String?
  region      String   // 地域
  level       String   // 层次：985/211/双一流/普通

  website     String?
  createdAt   DateTime @default(now())

model CampInfo {
  id            String   @id @default(uuid())
  title         String
  universityId  String
  collegeName   String?  // 所属学院
  majorIds      String   // JSON数组
  description   String?  @db.Text
  requirements  String?  @db.Text
  schedule      String?  @db.Text
  deadline      DateTime
  sourceUrl     String
  attachmentUrl String?
  status        String   @default("active")
  createdAt     DateTime @default(now())
  updatedAt     DateTime @updatedAt
  
  university    University @relation(fields: [universityId], references: [id])
  
  // 索引（评审建议）
  @@index([universityId, status])
  @@index([deadline, status])
  @@index([createdAt])
}

// 用户表（openid加密）
model User {

  nickname      String?
  avatarUrl     String?
  createdAt     DateTime @default(now())
  updatedAt     DateTime @updatedAt
  
  selections    UserSelection[]
  
  @@index([openid])
}
